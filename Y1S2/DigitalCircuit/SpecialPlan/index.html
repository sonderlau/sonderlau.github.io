# RS 触发器

异步触发器

由与非门组成的基本 RS 触发器

由两个稳定状态

  • 置位 set
    • 此时记忆二进制数1
  • 复位reset
    • 此时记忆二进制数0

逻辑符号：

加非说明 低电平有效

特性表：

优点

  • 电路简单 是构成各种触发器的基础

缺点

  • 输出受输入信号的直接控制 不能定时控制
  • 有约束条件
    • RS=0RS = 0RS=0
