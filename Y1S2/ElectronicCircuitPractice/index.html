# 分析误差产生的原因

  • 二极管在电路连通时会分压 而在理论计算中不考虑此分压
  • 电源接入处和电流输出处有较大的电阻
  • 万用电表测量精度有限 且会有一定的误差
  • 滤波电容等电路元件与标值有出入

# 请说明可调电源电路中接入的各元器件的作用 并简述稳压的工作流程

  • R4R_4R4​
    • 限流电阻
  • Q1Q2Q_1 \enspace Q_2Q1​Q2​
    • 三极管用于对电路的变化产生负反馈
    • 使电路的输出电阻趋于 0 电压可调且稳定
    • 双三极管使得负反馈更加精细
  • C21C22C_{21} \enspace C_{22}C21​C22​
    • 滤波电容
    • 降低输出电压的波动
  • Rp1R_{p1}Rp1​
    • 阻值可调的电阻
  • DzD_zDz​
    • 稳压二极管
    • 使电路的基准稳压平稳
  • R5R6R_5 \enspace R_6R5​R6​
    • 采样电阻
    • 将变化的电压获取一部分
  • R3R_3R3​
    • 与DzD_zDz​ 组成基准稳压部分
    • 提供稳定的基准稳压

工作过程 :

出于某些原因（电源电压波动或负载电阻的变化等）使输出电压升高（或降低）由Rp1R5R6R_{p1} \enspace R_5 \enspace R_6Rp1​R5​R6​ 组成的采样电路获取一部分改变的电压UFU_FUF​ 送到 Q3Q_3Q3​ 的基级

由 DZD_ZDZ​ 与R4R_4R4​ 组成 向Q3Q_3Q3​ 提供稳定的基准电压 UzU_zUz​

由Q3Q_3Q3​ 进行 UFU_FUF​ 与 UzU_zUz​ 的差值比较并放大到 Q2Q_2Q2​ 的基极

由 Q2Q1Q_2 \enspace Q_1Q2​Q1​ 对电压的改变做出负反馈调整
