# 6.1.2 有向图的连通性

  • 在有向图中,通路、回路、通路的长度、简单通路、简单回路、基本通路、基本回路等概念与无向图中相应概念非常类似,只要注意有向边方向的一致性即可。

  • 设D=<V,E>D=<V, E>D=<V,E>是有向图。∀vi,vj∈V\forall v_i,v_j\in V∀vi​,vj​∈V,若从viv_ivi​到vjv_jvj​存在通路,则称viv_ivi​可达viv_ivi​,记作vi→vjv_i\rightarrow v_jvi​→vj​。若vi→vjv_i\rightarrow v_jvi​→vj​,且vj→viv_j\rightarrow v_ivj​→vi​,则称viv_ivi​与vjv_jvj​是相互可达的,记作vi↔vjv_i\leftrightarrow v_jvi​↔vj​。规定 v,总是可达自身的,即vi↔viv_i\leftrightarrow v_ivi​↔vi​。

  • 设D=<V,E>D=<V, E>D=<V,E>是有向图

    1. 若∀vi,vj∈V\forall v_i,v_j\in V∀vi​,vj​∈V,都有vi↔vjv_i\leftrightarrow v_jvi​↔vj​,则称DDD为强连通的,否则称DDD为非强连通的。非强连通图中的极大强连通子图称为强连通分图；

    2. 若∀vi,vj∈V\forall v_i,v_j\in V∀vi​,vj​∈V都有vi→vjv_i\rightarrow v_jvi​→vj​或vj→viv_j\rightarrow v_ivj​→vi​,则称 D 为单向连通的,否则称DDD为非单向连通的。非单向连通图中的极大单向连通子图称为单向连通分图；

# 6.2 根 树

# 6.2.1 基本概念

  • 设TTT为有向树,若T 中有一个顶点的入度为 0,其余的顶点的入度均为 1,则称TTT为根树。入度为 0 的顶点称为根结点,又称树根,入度为 1 的顶点称为子结点。出度为 0 的顶点称为树叶,出度不为 0 的顶点称为分支点。既不是树叶又不是树根的顶点称为内点。从树根到任意顶点vvv的路的长度称为vvv的层数,层数最大的顶点的层数称为树高。将平凡图也看成根树,称为平凡树。
  • 设TTT为非平凡根树,∀vi,vj∈V\forall v_i,v_j\in V∀vi​,vj​∈V,若viv_ivi​可达vjv_jvj​,则称viv_ivi​为vjv_jvj​的祖先结点,vjv_jvj​为viv_ivi​的后代结点。若viv_ivi​邻接到vjv_jvj​（即<vi,vj>∈E<v_i,v_j>\in E<vi​,vj​>∈E）,则称viv_ivi​为vjv_jvj​的父结点（父亲）,vjv_jvj​为viv_ivi​的子结点（儿子）。
  • 若vi,vkv_i,v_kvi​,vk​ 的父结点(父亲)相同,则称 viv_ivi​ 与 vkv_kvk​ 是 兄弟。∀v∈V\forall v\in V∀v∈V,称顶点vvv及其后代结点的导出子图为以vvv为根的根子树。

# 6.2.2 二叉搜索树

  • 二叉搜索树是一种二叉树,它对应于某个有序集合。有序集合里的数据都存放在二叉树的顶点之中,使得对于树中的任意顶点vvv,vvv的左子树中的任意数据都比vvv中的数据小,而vvv的右子树中的任意数据都比vvv中的数据大。
