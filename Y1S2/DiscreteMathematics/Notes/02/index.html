# 离散数学

# 第二章 谓词逻辑

# 2.1 个体词、谓词与量词

# 2.1.1 个体词与谓词

  • 在原子命题中，表示对象的词称为个体词；表示对象所具有的性质或多个对象之间关系的词称为谓词。
  • 个体词一般是原子命题中的主语或宾语。个体词可以是具体事物，也可以是抽象的概念，例如，小王，夏天，偶数，思想等都可以作为个体词。特定的个体词称为个体常元，用小写字母 a,b,c,…a , b , c ,\dotsa,b,c,… 或 a1，a2，a3,…a_{1} ，a_{2}，a_{3},\dotsa1​，a2​，a3​,… 表示不确定的个体词称为个体变元，用小写字母 x,y,z,…x , y , z ,\dotsx,y,z,… 或 x1，x2，x3,…x_{1} ，x_{2}，x_{3},\dotsx1​，x2​，x3​,…表示。
  • 谓词一般是原子命题中的谓语，通常用大写字母P,Q,RP,Q,RP,Q,R 或 P1，P2，P3,…P_{1} ，P_{2}，P_{3},\dotsP1​，P2​，P3​,…表示。含有 nnn 个 (n≥0)(n\geq 0)(n≥0) 个体变元的谓词称为nnn 元谓词，也称为nnn 元简单命题函数，通常记为 P(x1，x2，x3,…，xn)P( x_{1} ，x_{2}，x_{3},\dots ，x_{n})P(x1​，x2​，x3​,…，xn​)。 它实际上是D1×D2×⋯×Dn⏞nintotal\overbrace{D_{1}\times D_{2}\times \cdots\times D_{n}}^{n in total}D1​×D2​×⋯×Dn​​nintotal​ 到 {0，10，10，1} 的一个函数，其中DiD_{i}Di​是个体变元xix_{i}xi​的个体域。所谓个体域就是个 体变元遍历的非空集合。一般地，个体域应事先给定，如果没有给定，则约定个体域是全体事物构成 的集合，称为全总个体域。另外，以后除非特别声明，否则认为一个nnn元谓词的所有个体变元的个体域 是一样的。
  • 由简单命题函数和命题连接词构成的表达式称为复合命题函数。
  • n 元谓词 (n≥0)( n\geq 0)(n≥0)不是命题，只有当nnn元谓词中的全部个体变元在个体域中取定个体常元后它才成为命题，此时，谓词已经不含个体变元而只含个体常元。通常，我们将不含个体变元的谓词称为 0 元谓词，比如P(a),Q(b,c),R(d,e,f)P(a),Q(b,c),R(d,e,f)P(a),Q(b,c),R(d,e,f)等都是000元谓词。0 元谓词是命题，命题逻辑中的命题都可表示为 0 元谓词，因此可将命题看成特殊的谓词。

# 2.1.2 量词

设DDD为量词，

“对DDD中的任意个体xxx”称为DDD上的全称量词，记为∀x\forall x∀x；

“对DDD中的存在个体xxx”称为DDD上的存在量词，记为∃x\exist x∃x。

# 2.2 谓词公式及其解释

# 2.2.1 谓词公式

  • 设Di(1⩽i⩽n)D_{i}(1\leqslant i\leqslant n)Di​(1⩽i⩽n)是相应于个体变元xix_{i}xi​的个体域，则相应于DiD_{i}Di​的项是指按下列
    规则定义的符号串：

    1. DiD_{i}Di​中的个体常元和个体变元是相应于DiD_{i}Di​的项；
    2. 若fff是从D1×D2×⋯×DnD_{1}\times D_{2}\times \cdots\times D_{n}D1​×D2​×⋯×Dn​到DiD_{i}Di​的nnn元函数，ti(1⩽i⩽n)t_{i}(1\leqslant i\leqslant n)ti​(1⩽i⩽n)是相应于DiD_{i}Di​的项，则f(t1,t2,t3,…,tn)f(t_{1},t_{2},t_{3},\dots,t_{n})f(t1​,t2​,t3​,…,tn​)也是相应于DiD_{i}Di​的项；
    3. 所有相应于DiD_{i}Di​的项都是有限次使用 1. 2. 得到的符号串。
    • 例如，设fff和ggg分别表示一元函数和二元函数，aaa是个体常元，xxx,yyy是个体变元，则a,x,y,f(x),g(x,y)，g(f(x),x)，f(g(x,y))a, x,y, f(x), g(x, y)，g(f(x), x)，f(g(x, y))a,x,y,f(x),g(x,y)，g(f(x),x)，f(g(x,y))等都是项。

  • 设P(x,x2,x3,…,xn)P(x, x_{2}, x_{3},\dots ,x_{n})P(x,x2​,x3​,…,xn​)是nnn元谓词,t(1≤i≤n)t(1\leq i\leq n)t(1≤i≤n)是相应于个体变元xix_{i}xi​的个体域
    DiD_{i}Di​的项，则称P(t,t2,t3,…,tn)P(t, t_{2}, t_{3},\dots ,t_{n})P(t,t2​,t3​,…,tn​)为原子谓词公式，简称原子公式。

    • 例如，设R(x,y,z)R(x, y, z)R(x,y,z)是三元谓词，z，f(z)，g(x,y)z，f(z)，g(x, y)z，f(z)，g(x,y)是三个项，则R(z,f(z),g(x,y))R(z, f(z), g(x, y))R(z,f(z),g(x,y))就是一个原子公式。

  • 谓词公式是按下列规则定义的符号串：

    1. 000和111是谓词公式；
    2. 原子公式是谓词公式；
    3. 若A，BA，BA，B是谓词公式，则¬A，¬B，A∧B,A∨B,A→B和A↔B\neg A，\neg B，A\land B, A\lor B, A\rightarrow B和A\leftrightarrow B¬A，¬B，A∧B,A∨B,A→B和A↔B是谓词公式；
    4. 若xxx是个体变元，AAA是谓词公式，则∀xA\forall xA∀xA和∃xA\exist xA∃xA是谓词公式；
    5. 所有的谓词公式都是有限次使用 1. 2. 3. 4. 得到的符号串。

  • 谓词公式是由原子公式、逻辑连接词、量词和圆括号等组成的符号串，命题逻辑中的命题公式仅是它的特例，所以命题逻辑包含于谓词逻辑之中。

  • 在谓词公式∀xA\forall xA∀xA和∃xA\exist xA∃xA中，称xxx为指导变元，称AAA为相应量词的辖域或作用域，辖域中凡与指导变元相同的个体变元称为约束变元，不是约束变元的个体变元称为自由变元。

  • 换名规则

    1. 在谓词公式中，将某量词辖域中出现的某个约束变元以及对应的指导变元改成本辖域中未曾出现过的个体变元符号，其余部分保持不变，公式的等价性不变。
    2. 在谓词公式中，将某个自由变元的所有出现用其中未曾出现过的某个体变元符号代替，其余部分保持不变，公式的等价性不变。

# 2.2.2 谓词公式的解释

  • 谓词公式AAA的一个解释III由下面四个部分组成：
    1. 非空个体域DDD；
    2. 对AAA中每个个体常元符号，指定DDD中一个固定元素；
    3. 对AAA中每个函数符号，指定一个具体的函数；
    4. 对AAA中每个谓词符号，指定一个具体的谓词。
  • 只有约束变元而没有自由变元的谓词公式称为封闭的谓词公式。
  • 设AAA是一个谓词公式，若AAA在任何解释下均为真，则称AAA为永真式（有效式）。若AAA在任何解释下均为假，则称AAA为永假式（矛盾式）。若存在解释使AAA为真，则称AAA为可满足式。
  • 设p,p2,…,pnp, p_{2},\dots ,p_{n}p,p2​,…,pn​是命题公式AAA中出现的nnn个命题变元，A,A2,…,AnA, A_{2},\dots ,A_{n}A,A2​,…,An​是nnn个谓词公式，用A(1≤i≤n)A(1\leq i\leq n)A(1≤i≤n)处处代替AAA中的PiP_{i}Pi​后所得谓词公式称为AAA的代替实例。
    • 例如，谓词公式P(x)→Q(x),∀P(x)→∃Q(x)P(x)\rightarrow Q(x),\forall P(x)\rightarrow \exist Q(x)P(x)→Q(x),∀P(x)→∃Q(x)等都是命题公式p→qp\rightarrow qp→q的代替实例，而∀x(P(x)→Q(x))\forall x(P(x)\rightarrow Q(x))∀x(P(x)→Q(x))不是p→qp\rightarrow qp→q的代替实例。
  • 永真式的代替实例是永真式，永假式的代替实例是永假式。

# 2.3 谓词公式的等价演算

  • 设AAA和 BBB 是两个谓词公式，如果在任何解释下，AAA和BBB都有相同的真值，则称AAA和BBB等价，记为A=BA=BA=B。
  • 设AAA和BBB是两个谓词公式，则A=BA=BA=B的充要条件是A→BA \rightarrow BA→B是永真式。

量词否定律

设AAA是命题公式，则：

  1. ¬∀xA(x)=∃x¬A(x)\neg \forall xA(x)=\exist x\neg A(x)¬∀xA(x)=∃x¬A(x)；
  2. ¬∃xA(x)=∀¬xA(x)\neg \exist xA(x)=\forall \neg xA(x)¬∃xA(x)=∀¬xA(x)。

量词辖域的收缩与扩展律

设A,BA,BA,B是谓词公式，BBB不含个体变元xxx，则：

  1. ∀x(A(x)∧B)=∀xA(x)∧B\forall x(A(x)\land B)=\forall xA(x)\land B∀x(A(x)∧B)=∀xA(x)∧B；
  2. ∀x(A(x)∨B)=∀xA(x)∨B\forall x(A(x)\lor B)=\forall xA(x)\lor B∀x(A(x)∨B)=∀xA(x)∨B；
  3. ∃x(A(x)∧B)=∃xA(x)∧B\exist x(A(x)\land B)=\exist xA(x)\land B∃x(A(x)∧B)=∃xA(x)∧B；
  4. ∃x(A(x)∨B)=∃xA(x)∨B\exist x(A(x)\lor B)=\exist xA(x)\lor B∃x(A(x)∨B)=∃xA(x)∨B。

量词分配律

设A,BA,BA,B是谓词公式，则有：

  1. ∀x(A(x)∧B(x))=∀xA(x)∧∀xB(x)\forall x(A(x)\land B(x))=\forall xA(x)\land \forall x B(x)∀x(A(x)∧B(x))=∀xA(x)∧∀xB(x)；
  2. ∃x(A(x)∨B(x))=∃xA(x)∨∃xB(x)\exist x(A(x)\lor B(x))=\exist xA(x)\lor \exist x B(x)∃x(A(x)∨B(x))=∃xA(x)∨∃xB(x)；

量词交换律

设A,BA,BA,B是谓词公式，则有：

  1. ∀x∀yA(x,y)=∀y∀xA(x,y)\forall x \forall y A(x,y)=\forall y \forall x A(x,y)∀x∀yA(x,y)=∀y∀xA(x,y)；
  2. ∃x∃yA(x,y)=∃y∃xA(x,y)\exist x \exist y A(x,y)=\exist y \exist x A(x,y)∃x∃yA(x,y)=∃y∃xA(x,y)。
  • 注意！！∀x∃yA(x,y)=∃y∀xA(x,y)\forall x \exist y A(x,y)=\exist y \forall x A(x,y)∀x∃yA(x,y)=∃y∀xA(x,y)是 错误 的!

置换规则

设ϕ(A)\phi (A)ϕ(A) 为含有子公式AAA的谓词公式，ϕ(B)\phi (B)ϕ(B)是用公式BBB置换ϕ(A)\phi(A)ϕ(A)中
的AAA(不要求处处置换)所得到的谓词公式，若A=BA=BA=B,则ϕ(A)=ϕ(B)\phi(A)=\phi(B)ϕ(A)=ϕ(B)。

# 2.4 谓词公式的推理演算

# 2.4.1 基本概念

  • 设A1,A2,…，An，BA_{1},A_{2},\dots，A_{n}，BA1​,A2​,…，An​，B是谓词公式，如果对A1,A2,…，AnA_{1},A_{2},\dots，A_{n}A1​,A2​,…，An​都取值 1 的任何解释，BBB必取值 1,则称由前提A1,A2,…，AnA_{1},A_{2},\dots，A_{n}A1​,A2​,…，An​到结论BBB的推理是有效的(正确的)，或者称BBB是
    前提A1,A2,…，AnA_{1},A_{2},\dots，A_{n}A1​,A2​,…，An​的逻辑结论(有效结论)，记为A1,A2,…，An⇒BA_{1},A_{2},\dots，A_{n}\Rightarrow BA1​,A2​,…，An​⇒B(A1∧A2∧⋯∧An⇒B)(A_{1}\land A_{2}\land \dots\land A_{n}\Rightarrow B)(A1​∧A2​∧⋯∧An​⇒B)。
  • 设AAA,BBB是两个谓词公式，则A=BA= BA=B的充要条件是A⇒BA\Rightarrow BA⇒B且B⇒AB\Rightarrow AB⇒A。
  • 设AAA,BBB是两个谓词公式，则A⇒BA\Rightarrow BA⇒B 的充要条件是A→BA\rightarrow BA→B是永真式。

# 2.4.2 演绎推理方法

  • 前面我们说过，判断一个谓词公式是否为已知前提的逻辑结论可以使用解释法和等价演算法，但这些方法的缺陷在于不能清晰地表达其推理过程。同命题逻辑一样，下面介绍运用等价公式、推理公式和推理规则的谓词逻辑演绎推理方法。
  • 谓词公式中含有量词使得谓词逻辑的演绎推理变得比命题逻辑的演绎推理要复杂。
  • 在谓词公式的演绎推理中，为了便于推理，有时需要引进或消去量词。因此在谓词公式的演绎推理中，除了要用到P 规则（前提引入规则）、E 规则（置换规则）和T 规则（结论引入规则）外，还要用到四条与量词有关的推理规则，它们在谓词公式的演绎推理方法中起着重要作用。

谓词公式推理规则

US 规则(全称量词消去规则)：∀xA(x)⇒A(a)or∀xA(x)⇒A(y)\forall xA(x)\Rightarrow A(a) \ or\ \forall xA(x)\Rightarrow A(y)∀xA(x)⇒A(a)or∀xA(x)⇒A(y).其中，yyy不在A(x)A(x)A(x)中以约束变元的形式出现。

ES 规则(存在量词消去规则)：∃xA(x)⇒A(a)\exist xA(x)\Rightarrow A(a)∃xA(x)⇒A(a)。 其中，个体常元aaa是使A(x)A(x)A(x)为真的特定的个体常元，这就要求 ①a 不在A(x)A(x)A(x)和已经推导出的公式中出现，② 除xxx外，A(x)A(x)A(x)中无其他自由变元。

UG 规则(全称量词引入规则)：A(y)⇒∀xA(x)A(y)\Rightarrow \forall x A(x)A(y)⇒∀xA(x)。其中，xxx不在A(y)A(y)A(y)中以约束变元的形式出现。

EG 规则(存在量词引入规则)：A(y)⇒∃xA(x)orA(a)⇒∃xA(x)A(y)\Rightarrow \exist x A(x)\ or\ A(a)\Rightarrow \exist x A(x)A(y)⇒∃xA(x)orA(a)⇒∃xA(x)。其中，xxx不在A(y)A(y)A(y)和A(a)A(a)A(a)中以约束变元的形式出现。
