# 关系模式

完整的关系模式定义

R(U,D,dom,F)R(U,D,dom,F) R(U,D,dom,F)

  • R
    • 关系名
  • U
    • 所有属性名的集合
  • D
    • 是 U 中属性所来自的域的集合
  • dom
    • 为属性向域映像的集合

# 连接 Join

是从两个关系的广义笛卡尔积中选取满足某规定条件的全体元组形成一个新的关系

R⋈AθBS=σAθB(R×S)R \underset{A \theta B}{\bowtie} S = \sigma_{A\theta B} (R \times S) RAθB⋈​S=σAθB​(R×S)

A B 不一定同名 但一定要可比 即来自同一个域

# 等值连接 Equivalence Join

所有的θ\thetaθ都是 = 的时候

这是较常用的连接方式

# 自然连接 Natural Join

要求相等的分量必须具有共同属性名

会把重复的 2 个属性名删去一个

R⋈SR \bowtie S R⋈S

# 外连接 Outer Join

如果把舍弃的元组也保存在结果关系中 而在其他属性上填null 这种连接叫外连接

  • 左外连接
    • R ⟕ S
  • 右外连接
    • R ⟖ S
  • 全外连接
    • R ⟗ S
